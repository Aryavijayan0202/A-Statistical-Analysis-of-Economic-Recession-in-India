# A Statistical Analysis of Economic Recession in India

This repository contains my MSc Statistics dissertation, which presents a comprehensive
statistical analysis of economic recession in India using time series, regression,
and clustering techniques.

## Objectives
- To model and forecast India’s Gross Domestic Product (GDP) using ARIMA time series models
- To examine the impact of major economic sectors on GDP using multiple regression analysis
- To classify Indian states based on economic indicators using K-means clustering

## Data Source
- Reserve Bank of India (RBI)
  - Handbook of Statistics on the Indian Economy
  - Sector-wise and state-wise economic data

## Methodology
- ARIMA time series analysis for GDP forecasting (1965–2023)
- Multiple linear regression for sector-wise GDP analysis (2011–2022)
- K-means cluster analysis for state-wise economic classification (2022–2023)

## Tools Used
- R
- Microsoft Excel
